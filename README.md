# Jimi Theme

A light color theme for Visual Studio Code, with an intentional **dark chrome frame** (activity bar, title bar, status bar) wrapping a light working area (editor, side bar, panels).

## Install

Available on:

- [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=grahampfy.jimi-theme)
- [Open VSX](https://open-vsx.org/extension/grahampfy/jimi-theme)

1. Open the Extensions view (`Ctrl+Shift+X`).
2. Search for **jimi-theme**.
3. Install, then select it via `Ctrl+K Ctrl+T` (Color Theme).

## Design

- **Light content** — editor, side bar, and panels use a light palette for readability.
- **Dark chrome** — the outer frame stays dark navy to keep the workspace boundaries clear.
